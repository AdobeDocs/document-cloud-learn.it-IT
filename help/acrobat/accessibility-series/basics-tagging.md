# Nozioni di base sui tag

Scopri le nozioni di base su come assegnare tag a un file PDF per renderlo accessibile. Gli argomenti trattati includono: correzione di elenchi, artefatti, aggiunta di testo alternativo per le immagini e tabelle semplici.

>
>Alcune funzioni di questo tutorial sono disponibili solo in Acrobat Pro, Premium e Studio.
