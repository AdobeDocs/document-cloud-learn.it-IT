# Protect un file PDF con una password

Scopri come [aggiungere una password al tuo PDF](https://www.adobe.com/it/acrobat/online/password-protect-pdf.html){target="_blank"} per impedire ad altri di aprire o [modificare il file PDF](https://www.adobe.com/it/acrobat/online/pdf-editor.html){target="_blank"}. La protezione dei file PDF impedisce l&#39;accesso non autorizzato, protegge le informazioni riservate, mantiene l&#39;integrità dei documenti e consente di rispettare determinate normative di settore.

>[!SLIDE](protect-pdfs)

>[!TIP]
>
>Per proteggere con password più file PDF, prova a utilizzare l&#39;[Action Wizard](../advanced-tasks/action.md).
